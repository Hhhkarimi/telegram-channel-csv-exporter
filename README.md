# Telegram Channel CSV Exporter

ابزار ساده و کاربردی برای استخراج پست‌های کانال‌های عمومی تلگرام در یک بازهٔ تاریخی مشخص و ذخیرهٔ خروجی به‌صورت فایل CSV، بدون نیاز به Telegram API.

این پروژه برای کسانی مناسب است که می‌خواهند متن پست‌ها، تاریخ انتشار، تعداد بازدید و لینک پست‌های یک کانال عمومی را برای تحلیل، آرشیو یا کار پژوهشی استخراج کنند.

---

## قابلیت‌ها

- استخراج پست‌های کانال عمومی تلگرام بدون API رسمی تلگرام
- خروجی CSV با پشتیبانی مناسب از متن فارسی در Excel
- پشتیبانی از تاریخ شمسی و میلادی
- پشتیبانی از اعداد فارسی، عربی و انگلیسی در ورودی تاریخ
- سه حالت استفاده:
  - رابط گرافیکی ساده با Tkinter
  - حالت تعاملی در ترمینال
  - حالت خط فرمان برای اسکریپت‌نویسی
- استفاده از `tgstat.com` به‌عنوان منبع اولیه
- فال‌بک خودکار به پیش‌نمایش وب تلگرام `t.me/s/` در صورت کپچا، بلاک یا خطای tgstat
- امکان تنظیم delay برای جلوگیری از rate-limit در بازه‌های طولانی

## نصب و اجرا

### 1. دریافت پروژه

```bash
git clone https://github.com/YOUR_USERNAME/telegram-channel-csv-exporter.git
cd telegram-channel-csv-exporter
```

### 2. ساخت محیط مجازی

#### Windows

```bash
python -m venv .venv
.venv\Scripts\activate
```

#### macOS / Linux

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

## روش‌های استفاده

### روش ۱: رابط گرافیکی

```bash
python tg_export_gui.py
```

در پنجرهٔ برنامه، نام یا لینک کانال، تاریخ شروع و تاریخ پایان را وارد کنید. سپس روی دکمهٔ استخراج بزنید و محل ذخیرهٔ فایل CSV را انتخاب کنید.

نمونه ورودی‌ها:

```text
irancurrency
https://t.me/irancurrency
@irancurrency
```

---

### روش ۲: حالت تعاملی ترمینال

```bash
python tg_channel_export.py
```

اگر برنامه را بدون آرگومان اجرا کنید، خودش مرحله‌به‌مرحله نام کانال، تاریخ شروع، تاریخ پایان و نام فایل خروجی را می‌پرسد.

---

### روش ۳: خط فرمان

```bash
python tg_channel_export.py irancurrency --from 1405-03-11 --to 1405-04-09
```

یا با لینک کامل کانال:

```bash
python tg_channel_export.py https://t.me/irancurrency --from 2026-06-01 --to 2026-06-30 -o out.csv
```

استفاده مستقیم از پیش‌نمایش وب تلگرام:

```bash
python tg_channel_export.py irancurrency --from 2026-06-01 --to 2026-06-30 --source tme
```

افزایش فاصله بین درخواست‌ها برای بازه‌های طولانی:

```bash
python tg_channel_export.py irancurrency --from 2026-01-01 --to 2026-06-30 --delay 2
```

## فرمت تاریخ

برنامه هم تاریخ شمسی و هم تاریخ میلادی را قبول می‌کند.

نمونه‌های معتبر:

```text
1405-04-01
۱۴۰۵/۰۴/۰۱
2026-06-22
2026/06/22
```

نکته: اگر سال کمتر از 1700 باشد، برنامه آن را شمسی در نظر می‌گیرد.

## ستون‌های خروجی CSV

| ستون | توضیح |
|---|---|
| `message_id` | شناسه عددی پست |
| `datetime_utc` | تاریخ و ساعت انتشار بر اساس UTC |
| `text` | متن پست |
| `views` | تعداد بازدید، در صورت وجود |
| `link` | لینک مستقیم پست |

فایل CSV با انکودینگ `utf-8-sig` ذخیره می‌شود تا متن فارسی در Excel درست نمایش داده شود.

## محدودیت‌ها

- فقط کانال‌های عمومی قابل استخراج هستند.
- اگر کانال خصوصی، محدودشده یا دارای محتوای غیرقابل مشاهده در وب باشد، برنامه نمی‌تواند آن را استخراج کند.
- ممکن است tgstat به‌دلیل محدودیت یا کپچا پاسخ ندهد؛ در این حالت برنامه به‌صورت خودکار سراغ `t.me/s/` می‌رود.
- برای بازه‌های خیلی طولانی، بهتر است مقدار `--delay` را افزایش دهید.
